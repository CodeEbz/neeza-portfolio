# Chinaza Ebenezer - Portfolio

A modern, responsive portfolio website built with React showcasing my projects and skills as a Full Stack Developer.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Modern Animations**: Smooth animations using Framer Motion
- **Project Showcase**: All major projects with live links
- **Skills Display**: Interactive skills grid
- **Contact Information**: Easy ways to get in touch
- **Social Links**: Direct links to GitHub, LinkedIn, and Twitter

## Technologies Used

- React 18
- Framer Motion (animations)
- React Icons
- CSS3 with modern features
- Responsive Grid Layout

## Projects Featured

1. **Remind To-Do App** - Task management with reminders
2. **Lebon Facilities** - Facility management platform
3. **Calculator App** - Interactive calculator
4. **Tech Dev Social** - Collaborative developer platform
5. **Ticket Generator** - Event ticket generation system

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```
